🤖 Web Chat Widget Flow

น้องใจดี · Spender Club · spenderclub.com → Vercel API → Gemini + Google Sheet

💬 ผู้ใช้พิมพ์ข้อความบน spenderclub.com
📦 widget.js (ฝังใน WordPress)
สร้าง Session ID จาก sessionStorage (ปิดแท็บ = หาย)
ส่ง POST /api/web-chat → Vercel
Body: { sessionId, message }
🛡 Anti-spam Checks
Honeypot — field _hp ถูกกรอก → เงียบทันที
IP Rate Limit — >20 ข้อความ / 5 นาที (ต่อ IP) → แจ้งเตือน
Session Blocked — โดน block จาก OOD → เงียบ
Session Rate Limit — >5 ข้อความ / 10 วินาที → "รอสักครู่นะคะ 🙏"
🚫 Nonsense Filter
ตรวจจับ keyboard mashing ก่อนส่ง Gemini:
· ข้อความ < 2 ตัวอักษร
· ตัวอักษรซ้ำทั้งหมด (aaa, ???)
· อักขระซ้ำ 3+ ครั้ง (ดดด, ไไไไ)
· พยัญชนะซ้ำ + วรรณยุกต์ (ทท่)
· พยัญชนะ + สระอำ + สระนำ (ฉำเ...)
→ "มีอะไรให้น้องใจดีช่วยไหมคะ 😊" ไม่ส่ง Gemini
🕐 Schedule Check — Web Chat (แยกจาก LINE OA และ Facebook)
อ่านกฎเวลาจาก Redis key schedule:rules:web
ตั้งได้ที่ /admin/schedule → แท็บ 🟠 Web Chat
ถ้าไม่มีกฎ = ตอบ 24 ชม.
ถ้าตั้งกฎและอยู่นอกเวลา → ตอบ "อยู่นอกเวลาทำการ inbox Facebook ได้เลยค่ะ" (ไม่เงียบ)
🙋 ลูกค้าขอแอดมิน? (shouldHandoff)
ตรวจจับคำ เช่น "ขอคุยแอดมิน", "ติดต่อคน", "อยากคุยกับเจ้าของ"
→ บันทึก history → ตอบ FALLBACK_MSG ทันที
🧠 Gemini 2.5 Flash (timeout 10 วินาที)
Input:
  • ข้อความจากลูกค้า
  • ประวัติบทสนทนา 20 ข้อความล่าสุด (Redis TTL 30 นาที)
  • FAQ จาก Google Sheet (cache 60 วินาที)
  • System Prompt + Hallucination Guard
แหล่งข้อมูลที่ Gemini ใช้
📊 Google Sheet (FAQ)
header: category, question, answer
เจ้าของร้านแก้ได้ตลอด
bot รีโหลดทุก 60 วินาที
🤖 Gemini Knowledge
ใช้เมื่อ FAQ ไม่มีข้อมูล
ถ้าไม่มั่นใจ → ตอบ NOT_FOUND
ไม่เดาหรือ hallucinate
🔀 ผลลัพธ์จาก Gemini
HANDOFF / GEMINI_UNAVAILABLE บอทตัดสินใจส่งต่อ หรือ timeout/429/503
→ ตอบ FALLBACK_MSG
OUT_OF_DOMAIN คำถามไม่เกี่ยวกับวิทยุสื่อสาร
ครั้งที่ 1 → แจ้งขอบเขต
ครั้งที่ 2–4 → เงียบ
ครั้งที่ 5+ → block session 30 นาที
NOT_FOUND ไม่มีข้อมูลในระบบ
→ บันทึก unanswered_log
→ ตอบ FALLBACK_MSG ทันที
ตอบปกติ ✅ ตรวจ Duplicate:
ตอบซ้ำ → reminder
ตอบซ้ำ 2+ → เงียบ
บันทึก question_freq
📨 ส่ง reply กลับ widget.js
JSON: { "reply": "ข้อความตอบกลับ" }
widget แสดงข้อความในกล่องแชท · URL ในข้อความกดได้ (linkify)
บันทึก history ลง Redis (TTL 30 นาที)
✅ ลูกค้าเห็นข้อความตอบกลับ
📢 FALLBACK_MSG — ข้อความ fallback สำหรับทุกกรณีที่ตอบไม่ได้ใน Web Chat
"รบกวนสอบถามรายละเอียดเพิ่มเติมได้ที่ @spenderclub โทร 085-222-9111 ค่ะ"
ใช้เมื่อ: NOT_FOUND · shouldHandoff · HANDOFF signal · GEMINI_UNAVAILABLE (timeout/429/503)

🗄 Redis Keys ที่ใช้

Keyหน้าที่TTL
webchat:hist:{sid}ประวัติบทสนทนา (max 20 ข้อความ)30 นาที
webchat:rate:{sid}Session rate limit นับจำนวนข้อความ10 วินาที
web_ip_rate:{ip}IP rate limit (max 20/5นาที)5 นาที
web_blocked:{sid}Session ถูก block จาก OOD ≥5 ครั้ง30 นาที
webchat:nonsense:{sid}นับ out-of-domain ต่อเนื่อง10 นาที
webchat:last_answer:{sid}ป้องกันตอบซ้ำ2 นาที
webchat:repeat:{sid}นับการตอบซ้ำ2 นาที
unanswered_logบันทึกคำถามที่บอทตอบไม่ได้ (max 500)ถาวร
question_freqSorted set สถิติคำถามยอดนิยมถาวร
