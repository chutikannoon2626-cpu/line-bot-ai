ตัวอย่าง: ถ้าเห็น "ขอดูคู่มือ TC-5M" บ่อย → เพิ่มแถวใหม่ใน Sheet:

4. Tab 🔥 ถามบ่อย

เมื่อไหรที่บันทึก: ทุกครั้งที่บอทตอบสำเร็จ (ไม่ใช่ตอบไม่ได้) ระบบจะนับ +1 ให้คำถามนั้น


ข้อมูลรายละเอียด
อันดับ 1–3badge สีเข้ม (navy)
อันดับ 4+badge สีเทา
จำนวนครั้งนับรวม LINE OA + Facebook + Web Chat
แสดงสูงสุด30 อันดับ

📌 ข้อมูลสะสมจากวันที่ deploy เป็นต้นไป — ไม่มีข้อมูลย้อนหลัง · ปุ่ม ⬇ Export Excel (2026-07-07 เปลี่ยนจาก CSV) · กด ล้างสถิติ เพื่อเริ่มนับใหม่

ประโยชน์: รู้ว่าลูกค้าสนใจสินค้า/บริการอะไรมากที่สุด → นำไปปรับปรุง Google Sheet หรือออกโปรโมชั่นได้ตรงจุด

5. Tab 💬 ประวัติแชท

เมื่อไหรที่บันทึก: ทุกข้อความที่ลูกค้าและบอทคุยกัน (ทั้งฝั่งลูกค้าและฝั่งบอท) ทั้ง 3 ช่องทาง


ข้อมูลรายละเอียด
รายการหลัก100 conversation ล่าสุด (เรียงตามข้อความล่าสุดของแต่ละคน)
Dropdown filterทุกช่องทาง / LINE OA / Facebook / Web
ตารางช่องทาง | ข้อความล่าสุด | จำนวน | เวลา | ⬇
กดแถวขยายดูบทสนทนาทั้งหมดแบบ chat bubble
⬇ ต่อแถวดาวน์โหลด Excel เฉพาะบทสนทนาของคนนั้น (ไม่จำกัดวัน สูงสุด 200 ข้อความล่าสุด)
⬇ ดาวน์โหลดวันนี้Excel รวมทุกช่องทางของวันนี้เท่านั้น (ไม่ผูกกับ dropdown filter)
⬇ ดาวน์โหลดทั้งหมด(2026-07-07 เพิ่มใหม่) Excel รวมทุกช่องทางทุกวัน — จำกัดตาม TTL 30 วัน + 200 ข้อความ/user ที่ระบบเก็บไว้จริง ไม่ใช่ไม่มีที่สิ้นสุด

6. ข้อมูลที่เก็บใน Redis

Keyประเภทเนื้อหาขนาดสูงสุด
unanswered_log List JSON: { ts, userId, question } 500 รายการ
question_freq Sorted Set member = คำถาม, score = จำนวนครั้ง ไม่จำกัด
chatlog:convs Sorted Set member = userId, score = เวลาข้อความล่าสุด ไม่จำกัด (TTL 30 วันต่อ user ผ่าน chatlog:u:*)
chatlog:u:{'{userId}'} List JSON: {'{'} userId, channel, role, message, ts {'}'} 200 ข้อความ/user · TTL 30 วัน
chatlog:uchan Hash userId → channel ไม่จำกัด

ดูได้โดยตรงที่ Upstash Dashboard → Data Browser เช่นกัน

7. ไฟล์โค้ดที่เกี่ยวข้อง

ไฟล์หน้าที่
app/api/admin/unanswered/route.tsGET ดึงข้อมูล, DELETE ล้างข้อมูล (auth ด้วย ADMIN_SCHEDULE_PASSWORD)
app/admin/unanswered/page.tsxหน้า UI 3 tabs: ตอบไม่ได้ / ถามบ่อย / ประวัติแชท · Export Excel ทั้ง 3 tab
app/api/admin/chatlog/route.tsGET รายการ 100 conversation ล่าสุด + messages ของ user คนหนึ่ง
app/api/admin/chatlog/export/route.tsGET สร้างไฟล์ Excel — รายวัน (date=) / ทั้งหมด (all=1) / รายคน (userId=)
lib/chatlog.tslogChat() บันทึกทุกข้อความเข้า chatlog:convs/chatlog:u:*/chatlog:uchan
app/api/line-webhook/route.tszincrby question_freq +1 หลังตอบ LINE สำเร็จ + logChat()
app/api/fb-webhook/route.tszincrby question_freq +1 หลังตอบ Facebook สำเร็จ + logChat()
app/api/web-chat/route.tszincrby question_freq +1 หลังตอบ Web Chat สำเร็จ + logChat()
